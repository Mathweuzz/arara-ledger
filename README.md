# AraraLedger — Core Bancário Educativo em COBOL

Projeto educativo de core bancário em COBOL usando GnuCOBOL e arquivos indexados (ISAM),
sem banco SQL. O objetivo é implementar plano de contas, lançamentos contábeis em partidas
dobradas, processamento em lote e relatórios contábeis.

## Requisitos

- GnuCOBOL
- make
- git
